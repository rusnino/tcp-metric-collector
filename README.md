# TCP Metric Collector



## Getting started

The TCP Metrics for a Linux box.

## Description
The Python script that collects TCP Metrics per destionation IP address. It should be run on sender side.
The script works for python 2.7 and python 3.x

## Installation
Just copy the script to a folder on local Linux box.

## Usage
python tcp_metrics_collector.py -a <Destination IP address>
